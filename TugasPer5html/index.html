1. Apa itu version control?

version control adalah sistem yang mencatat setiap perubahan pada sebuah yang dilakukan pada sekumpulan berkas dari waktu ke waktu, salah satu tools yang bisa di pakai yaitu Git

2. Perbedaan Git Dan Github

Git adalah sebuah Version Control System lokal yang berfungsi mencatat setiap perubahan pada codingan jadi kita bisa kembali ke versi sebelumnya jika terjadi kesalahan

GitHub adalah platform berbasis web yang menyediakan hosting untuk proyek yang menggunakan Git, GitHub bertindak sebagai penyimpanan di internet, mempermudah kolaborasi agar bisa di lihat oleh orang lain.

3. Apa itu HTML?

markup language yang digunakan untuk membuat dan menyusun halaman web
